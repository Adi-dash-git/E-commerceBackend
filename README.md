# 🛒 Sales Savvy – Modular E-Commerce Platform

A scalable, full-stack e-commerce platform built using **Java, Spring Boot, React.js, and MySQL**. The application follows a modular architecture with independent backend services for authentication, user management, product management, cart, orders, and payments.

## 🚀 Features

* 🔐 Secure JWT-based Authentication & Authorization
* 👥 Role-Based Access Control (Admin & Customer)
* 📦 Product Catalog Management
* 🛒 Shopping Cart Management
* 📋 Order Placement & Tracking
* 💳 Secure Payment Integration
* 📱 Responsive React.js Frontend
* 🔒 Password Encryption & Data Validation
* 📊 Optimized Database Design
* 🌐 RESTful API Architecture

## 🏗️ Project Architecture

The backend is divided into the following modules:

```
SalesSavvyBackend
│
├── Authentication Service
├── User Management Service
├── Product Service
├── Cart Service
├── Order Service
└── Payment Service
```

Each module is responsible for a specific business functionality, making the application easy to maintain, extend, and scale.
